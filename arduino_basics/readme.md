# Arduino Basics
Before starting the final Plant Monitoring System, several basic Arduino experiments were completed in order to understand the fundamentals of embedded systems programming, sensor integration, and hardware interaction.

These experiments served as proof-of-concept exercises and helped build familiarity with:
- digital and analog inputs/outputs
- sensor readings
- LED control
- serial communication
- hardware debugging

# Overview

## 01 - Basics : blink.ino

### Concepts
- Analog Read Serial
- Blink
- Digital Read Serial
- Fading an LED
- Read Analog Voltage
 
### Hardware

- 220Ω resistor
